# Frontend Documentation  
## Ireland Immigration Research Website

---

## 1. Project Overview

This project is an **academic web-based research platform** created to collect and visualize data related to **immigration to Ireland**.  
The website allows users to participate in a survey, submit their responses, and view aggregated results through a dashboard.

The main focus of this project is:
- Frontend structure  
- User experience (UX)  
- Ethical data collection  
- Team collaboration using Git  

---

## 2. Frontend Pages Structure

The frontend is composed of **three main pages**, representing the complete user journey.

### Pages Included
1. Landing Page (Survey Form)  
2. Confirmation Page  
3. Dashboard Page (Results Visualization)  

Each page is implemented as a separate HTML file.

---

## 3. Landing Page – Survey Form

**File:** `landing-page.html`

### Purpose
The landing page is responsible for **collecting user responses** for the research survey.

### Features
- Main research question:  
  *“Why did you choose to live in Ireland?”*
- Predefined answer options:
  - Learn English  
  - Job Opportunities  
  - Quality of Life  
  - Career / Future  
- Personal information fields:
  - Full Name  
  - City  
  - Nationality  
  - Email  
- Submit button to send the survey

### UX Considerations
- Simple and clean layout  
- Icons to support user understanding  
- Minimal text input to reduce friction  
- Responsive design for desktop and mobile  

---

## 4. Confirmation Page

**File:** `confirmation-screen.html`

### Purpose
This page confirms that the survey was successfully submitted.

### Features
- Clear success confirmation message  
- Thank-you message for the participant  
- Visual feedback (icons / confirmation text)  
- Navigation option to view results  

### UX Considerations
- Clear and immediate feedback  
- No additional data collection  
- Minimal and distraction-free design  

---

## 5. Dashboard Page – Results Visualization

**File:** `dashboard.html`

### Purpose
The dashboard presents **aggregated survey results** in a clear and visual format.

### Features
- Summary cards with key metrics  
- Charts showing:
  - Main reasons for immigration  
  - Age group distribution  
- Insights section for data interpretation  

### UX Considerations
- Readable charts and labels  
- High-contrast colors for clarity  
- Focus on insights rather than raw data  

---

## 6. Technologies Used

- HTML5  
- CSS (utility-based styling)  
- Icons for visual support  
- Vanilla JavaScript (planned for interaction logic)

No backend or database is connected at this stage.  
The project focuses on frontend structure and usability.
